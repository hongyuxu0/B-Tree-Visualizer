# B-、B+、B* 多路平衡查找树可视化交互工具
## 项目介绍
本项目基于 Python + Tkinter + Matplotlib 开发，完整实现三种经典多路平衡查找树：
1. **B-Tree**：标准多路平衡树，关键字均匀分布在索引与叶子节点
2. **B+ Tree**：数据库底层索引结构，叶子节点双向有序链表，范围查询高效
3. **B* Tree**：优化分裂策略，优先兄弟节点重分配，空间利用率、树高表现最优

### 核心两大模块
1. **可视化交互页**：全操作分步动画演示，单步回放、自动播放，直观展示节点插入溢出、删除下溢、借键、合并、分裂底层逻辑
2. **性能对比页**：批量自动生成 m=3~10 阶树，统计四类性能指标并生成对比图表，可直接用于课程实验报告

配套提供三套树独立 C++ 实现源码 + 完整中文使用说明文档，适合数据结构课程学习、实验作业、算法原理可视化验证。

## 可视化页面功能详解
### 1. 基础CRUD操作
- 插入：输入数字关键字，分步动画展示「查找路径→叶子插入→溢出分裂/重分配→上推分隔键」，禁止重复关键字
- 删除：查找目标关键字，内部节点自动替换前驱，叶子删除后处理下溢，支持向兄弟借键、节点合并
- 查找：蓝色高亮完整查找路径，找到节点绿色高亮，不存在标红提示
- 更新：底层逻辑为先删除旧键、再插入新键，完整回放两步操作动画

### 2. 最值、前驱、后继查询
- 最小值：自动遍历至最左侧叶子首关键字，高亮完整路径
- 最大值：自动遍历至最右侧叶子末尾关键字
- 查找前驱：获取小于目标键的最大关键字，无则提示无前驱
- 查找后继：获取大于目标键的最小关键字，无则提示无后继

### 3. 区间范围查询
输入左右边界，自动定位起始叶子节点，沿有序链表收集区间内全部关键字；
可直观对比：B+树仅遍历叶子链表，查询效率远高于B/B*树。

### 4. 遍历动画
支持四种遍历方式，分步动画展示遍历流程并输出有序结果：
1. 前序遍历：优先访问索引节点关键字，再递归子树
2. 中序遍历：B/B*树标准递归；B+树直接遍历叶子链表，输出严格升序序列
3. 后序遍历：先递归全部子树，最后输出当前节点关键字
4. 层序遍历：从上至下、从左到右层级遍历所有节点

### 5. 辅助校验工具
- 统计总数：动画遍历整棵树，累加输出全部关键字总数量
- 检查树合法：逐节点校验对应树的硬性数学约束，非法节点标红并输出错误原因，可用于作业校验构造正确性

### 6. 动画控制
- 下一步：单帧分步调试，适合拆解分裂、合并底层细节
- 上一步：回退上一帧动画，对比操作前后树结构变化
- 播放：自动连续播放完整操作流程，可拖动滑块调整动画切换速度（200~2000ms）
- 重置：清空所有动画历史，回到空白初始树

## 性能对比模块说明
### 测试参数
- 单棵树测试数据量：自定义单次插入随机关键字数量（推荐500~2000）
- 一键批量测试：自动循环生成 m=3,4,5...10 阶的 B/B+/B* 树，后台子线程运行不阻塞UI

### 四张性能图表指标
1. **左上：阶数m - 树高折线图**
树高越小，磁盘IO查询次数越少；B*树重分配机制树高最低，B树树高最高。
2. **右上：各阶总分裂次数柱状图**
分裂次数越少，磁盘写入开销越小；B*树优先兄弟借键，分裂次数显著少于B、B+树。
3. **左下：节点平均填充率折线图**
填充率 = 总关键字 ÷ 全部节点总容量，数值越高空间利用率越好，B*树填充率最优。
4. **右下：插入总键比较次数柱状图**
代表查找过程关键字对比计算开销，数值越低查询速度越快。

## 三种树核心原理差异
| 类型 | 数据存储位置 | 链表结构 | 分裂策略 | 适用场景 |
|------|------------|---------|---------|---------|
| B-Tree | 索引+叶子节点均存真实关键字 | 无有序叶子链表 | 节点溢出直接二分分裂 | 小型内存索引 |
| B+ Tree | 索引仅存分隔键，全部真实数据存叶子 | 叶子双向有序链表 | 仅拆分叶子，索引更新分隔键 | 数据库、文件系统索引 |
| B* Tree | 索引+叶子均存关键字 | 无链表 | 溢出优先向兄弟借键；填满后三分拆分三节点 | 海量持久化存储，追求高空间利用率 |

## 常见问题解决
1. Matplotlib图表中文显示方框□□□
Windows系统直接运行无问题；Linux安装文泉驿微米黑字体，Mac更换Arial Unicode MS字体。
2. 点击「使用帮助」提示文件不存在
将 `usehelp.txt` 与 `btree_visualizer.py` 放在同一项目根目录。
3. 性能测试点击按钮无响应
等待状态栏显示「完成」再发起新测试，避免多线程并发冲突。
4. 输入数字后插入无效果
输入框存在空格、中文、符号，清空输入框仅输入纯整数。

## 学习使用建议
1. 新手入门：点击「演示序列」一键插入标准测试数据，使用「播放」自动观看完整插入分裂动画。
2. 对比学习：同一阶数、同一组关键字，分别切换三种树，直观对比树高、分裂次数、填充率差异。
3. 课程实验：使用批量性能测试生成图表，直接作为实验报告数据支撑。
4. 原理验证：手动构造临界节点（刚好满、刚好下溢），通过插入/删除观察借键、合并逻辑。
